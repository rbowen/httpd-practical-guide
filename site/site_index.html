Apache httpd: A Practical Guide

by Rich Bowen

Solutions and examples for Apache HTTP Server administrators. Covers installation, virtual hosts, URL mapping, authentication, SSL/TLS, proxying, performance tuning, logging, security, dynamic content, and much more — in a practical recipe-based format.

Based on over twenty-five years of working with the Apache HTTP Server, answering questions on the users mailing list, and contributing to the project.
